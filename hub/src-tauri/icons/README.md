# Tauri icons

These are **placeholder** icons authored from scratch for the v3.1.0
development cycle:

| File | Author | Format | Notes |
|---|---|---|---|
| `icon.ico` | Generated by `hub/src-tauri/icons/regenerate.js` | 16×16 32-bit BGRA Windows ICO | dark grey opaque |
| `icon.png`, `32x32.png`, `128x128.png` | Same generator | 16×16 RGBA PNG | dark grey opaque |
| `icon.icns` | Same generator | (PNG bytes — NOT a valid ICNS yet) | replace before macOS builds |

Before v3.1.0 final ships:

1. Commission a 1024×1024 master PNG from the design team.
2. Run `pnpm dlx @tauri-apps/cli icon path/to/master-1024.png` from
   the `hub/` directory; it overwrites every file in this folder
   with properly-sized variants including a real ICNS.

Until then, `cargo check` and `cargo build` succeed because the
ICO is a valid 32-bit BGRA Windows icon — just visually a grey
square. The macOS build path will fail until the ICNS is replaced.
